자주 묻는 질문

GPT 포스트 팩토리는 어떤 서비스인가요?

GPT/Gemini 등 최신 AI 모델을 활용해 블로그에 최적화된 고품질 콘텐츠를 자동으로 생성하는 서비스입니다. 워드프레스, 티스토리, N사 블로그를 모두 지원합니다.

시스템 요구사항은 어떻게 되나요?

윈도우OS 10 이상(64bit), Chrome 최신 버전, 인터넷 연결이 필요합니다. Mac OS에서는 사용이 불가합니다.

서비스 제공 기간은 어떻게 되나요?

프로그램은 구매 후 영구적으로 사용 가능하며, 무료 A/S 보장 기간은 구매일로부터 1년입니다.

여러 PC에서 사용 가능한가요?

GPT 포스트 팩토리는 1제품키 1PC 원칙으로 판매됩니다. 추가 PC 사용 시 추가 구매가 필요하며, 프리미엄 패키지 구매 시 할인된 가격으로 추가 라이센스를 구매할 수 있습니다.

컴퓨터가 고장나거나 교체할 경우는 어떻게 되나요?

컴퓨터 교체 시 재등록할 수 있도록 도와드립니다.

윈도우 업데이트로 프로그램 사용이 불가능해질 수 있나요?

윈도우 업데이트로 인한 사용 불가 문제는 발생하지 않습니다.

체험판은 제공되나요?

현재 체험판은 제공되지 않습니다.

GPT 포스트 팩토리의 가격 정책과 결제 방법은 어떻게 되나요?

일회성 라이센스 구매 방식으로 제공되며, 추가 구독료 없이 영구적으로 사용 가능합니다. 다만, AI 모델(GPT/Gemini 등)의 API 사용에는 별도 비용이 발생할 수 있습니다.

스탠다드 구입 후 프리미엄으로 업그레이드 가능한가요?

네, 스탠다드 구매 후 추가 비용을 지불하면 프리미엄으로 업그레이드 가능합니다. 카카오 링크로 관리자에게 연락주세요.

티스토리만 사용하는 패키지도 있나요?

네, 티스토리만 사용 가능한 패키지는 구매 가능합니다. 카카오 링크로 관리자에게 연락주세요.

워드프레스만 되는 패키지를 구매한 후 나중에 네이버 블로그도 추가할 수 있나요?

네, 워드프레스만 되는 패키지 구매 후 추가 비용을 지불하면 패키지 업데이트가 가능합니다. 카카오 링크로 관리자에게 연락주세요.

블로그 계정은 몇 개까지 사용 가능한가요?

블로그 계정 개수에는 제한이 없습니다. 다만, 계정을 변경할 때마다 설정 메뉴에서 계정 정보를 변경하고 다시 글쓰기 시작 버튼을 눌러야 합니다.

2차 인증이 설정된 네이버 계정도 사용 가능한가요?

네이버는 백그라운드로 접속하며, 2차 인증은 한 번 확인이 필요합니다.

하루에 여러 계정에 각각 포스팅이 가능한가요?

네, 각 계정마다 글 작성이 가능합니다. 다만 계정 변경 시 설정 메뉴에서 계정을 변경하고 다시 글쓰기를 시작해야 합니다.

AI 모델은 어떤 것을 사용하나요?

ChatGPT와 Gemini 두 가지 AI 모델 중 선택 가능합니다.

글의 양 조정이 가능한가요?

네, 글의 양 조절이 가능합니다.

1포스팅당 API 비용은 얼마인가요?

요청하는 프롬프트에 따라 다르지만, 보통 10원 이하입니다.

생성된 콘텐츠의 상업적 사용이 가능한가요?

네, 생성된 모든 콘텐츠의 저작권은 사용자에게 귀속되어 상업적 용도로 자유롭게 활용 가능합니다. 단, AI 생성 콘텐츠의 활용과 관련된 모든 책임은 사용자에게 있습니다.

직접 작성한 글을 사용할 수 있나요?

완전히 동일한 글은 사용할 수 없으며, AI를 통한 재구성이 필요합니다.

글 작성 방식은 어떻게 되나요?

백그라운드 방식으로 작동하여 포스팅 누락 문제는 발생하지 않습니다.
